Miwotreff

MiWoTreff ist eine App, die das Programm des "Mittwochstreff der Evangelischen Gemeinschaft Bogenhausen, München" anzeigt und weitere Funktionen (Suche, Import, etc.) bietet. --- MiWoTreff is an app for showing and searching the schedule of the weekly meetings of the evangelic community of Bogenhausen, Munich.

MiWoTreff

Über / About

Die App MiWoTreff zeigt das Programm des "Mittwochstreff der Evangelischen Gemeinschaft Bogenhausen, München".

The app MiWoTreff shows the schedule of the weekly meetings of the evangelic community of Bogenhausen, Munich.

Installation

Zum Installieren der APK-Datei muss unter den Sicherheits-Optionen die Option "Installation von Apps aus unbekannten Quellen zulassen" aktiviert werden.

Dann einfach vom Smartphone / Tablet aus folgenden Link aufrufen und die Installation durchführen oder den QR Code scannen:

To install the apk file you have to activate the option "Allow installation of apps from unknown sources" under Security options.

Then click on the device (smartphone / tablet) the link below to install the app or scan the qr code to the link:

Version 6.3.0: miwotreff.apk

Kompatibilität / Compatibility

This app is compatible with Android 4.0 and above.
